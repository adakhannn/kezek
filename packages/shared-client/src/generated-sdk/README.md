# Generated API SDK (OpenAPI v1)

Эта директория зарезервирована для автоматически сгенерированного клиентского SDK поверх OpenAPI‑спеки (`/api/swagger.json`).  
Генерация выполняется командой из корня репозитория:

```bash
pnpm generate:sdk
```

Это:
- скачает текущую OpenAPI‑схему (`/api/swagger.json` — **нужен запущенный dev/prod сервер**);
- сгенерирует типы и функции клиента в `packages/shared-client/src/generated-sdk` (на базе `openapi-typescript-codegen` с клиентом `fetch`).

> Внимание: не редактируйте файлы внутри `generated-sdk` вручную — они будут перезаписаны при следующей генерации.

Интеграция с существующим `createApiClient` остаётся на усмотрение команды: можно либо использовать сгенерированный клиент напрямую, либо обернуть его адаптером для логирования/обработки ошибок.
