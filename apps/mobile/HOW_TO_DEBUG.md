# Как открыть консоль для отладки в Expo Go

## Способ 1: DevTools в Expo Go

### Android:
1. Встряхните телефон
2. Или нажмите кнопку меню (три точки)
3. Выберите "Debug Remote JS"
4. Откроется браузер с консолью разработчика

### iOS:
1. Встряхните телефон
2. Или нажмите `Cmd+D` в симуляторе
3. Выберите "Debug Remote JS"
4. Откроется браузер с консолью разработчика

## Способ 2: Логи в терминале

Логи из `console.log` также появляются в терминале, где запущен `npx expo start`.

Ищите строки:
```
=== Environment Variables Debug ===
=== Supabase Config Debug ===
```

## Способ 3: React Native Debugger

1. Установите React Native Debugger
2. В Expo Go выберите "Debug Remote JS"
3. Откроется отдельное окно с полной консолью

## Способ 4: Логи через adb (Android)

```bash
adb logcat | grep -i "react\|expo\|supabase"
```

## Что искать в логах

- `=== Environment Variables Debug ===` - показывает, какие переменные загружены
- `=== Supabase Config Debug ===` - показывает конфигурацию Supabase
- `Error:` - ошибки приложения
- `Warning:` - предупреждения

## Если консоль не открывается

1. Убедитесь, что телефон и компьютер в одной сети
2. Попробуйте перезапустить Expo Go
3. Проверьте, что DevTools не заблокированы настройками безопасности
