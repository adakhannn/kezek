# Авторизация в Expo

## Установка Expo CLI

Expo CLI обычно устанавливается автоматически через `npx expo`, но можно установить глобально:

```bash
npm install -g expo-cli
# или
pnpm add -g expo-cli
```

## Авторизация

### Вариант 1: Через команду (рекомендуется)

```bash
cd apps/mobile
npx expo login
```

Введите email и пароль от вашего Expo аккаунта (того же, что используется в Expo Go на телефоне).

### Вариант 2: Через браузер

```bash
npx expo login --web
```

Откроется браузер для авторизации.

### Проверка авторизации

```bash
npx expo whoami
```

Покажет ваш email, если вы авторизованы.

### Выход из аккаунта

```bash
npx expo logout
```

## После авторизации

После успешной авторизации запустите:

```bash
npx expo start --tunnel
```

Или:

```bash
npx expo start --clear
```

## Создание аккаунта Expo

Если у вас нет аккаунта:

1. Откройте [expo.dev](https://expo.dev)
2. Зарегистрируйтесь
3. Используйте тот же email в Expo Go на телефоне
4. Авторизуйтесь через `npx expo login`

## Важно

- Используйте **тот же аккаунт** в Expo CLI и в Expo Go на телефоне
- Это необходимо для работы с проектами в Expo Go
